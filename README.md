# CSS Odyssey

AI-assisted full-stack web app for learning CSS layout through game-like challenges.

## Stack
- Next.js
- TypeScript
- Tailwind CSS
- GSAP
- Supabase
- Hugging Face
- Vercel

## MVP
- CSS layout challenges
- Progress tracking
- Challenge paths
- AI hint feature
- Animated UI

## Scope for v1
- Flexbox
- Grid
- Responsive Design
